# Matchday MOD 1.4.0

[English](#english) | [简体中文](#简体中文)

## English

### Three commentary styles

- Adds the persistent `espn.commentary_style` preference with `casual`,
  `balanced`, and `professional` values. Existing configurations default to
  `balanced`.
- All three styles use the same parsed event facts and retain the match time,
  event, team, required players, result, and current score. Unconfirmed events
  remain explicitly uncertain.
- Professional speech can add reliably parsed ESPN details while keeping the
  device balloon compact. Missing or ambiguous details are not inferred, and
  raw English commentary is not read aloud.
- Styling covers ESPN events, match phases and results, plus Kalshi market
  jumps and suspected-goal alerts. It does not change reactions, priorities,
  alert switches, TTS voice, or TTS rate.

### Setup and compatibility

- Both the device `/setup` page and watcher-hosted setup page expose the
  three-way selector and current effective style.
- The watcher accepts `POST /api/setup/style`; the device accepts
  `POST /api/match-setup/style` and uses the existing pending/ack relay.
- Style changes apply immediately to newly generated alerts without replaying
  ESPN events or resetting market baselines, queued alerts, or polling state.

To upgrade, update the watcher checkout and install Matchday MOD 1.4.0. The
official Stack-chan host firmware and TTS module do not need to be changed or
reflashed for this release.

## 简体中文

### 三档播报语气

- 新增全局持久偏好 `espn.commentary_style`，可选 `casual`、`balanced` 和
  `professional`；旧配置默认继续使用 `balanced`。
- 三档共用同一份解析后的事件事实，都会保留比赛时间、事件、球队、必要球员、
  结果和当前比分；未确认事件在任何语气下都保持明确的不确定性。
- 专业档语音可补充从 ESPN 可靠解析出的细节，同时保持设备气泡精炼。缺失或含糊
  信息不会被推断，也不会直接朗读英文文字直播原文。
- 语气覆盖 ESPN 事件、比赛阶段与赛果，以及 Kalshi 盘口突变和疑似进球提醒；
  不改变动作、优先级、提醒开关、TTS 音色或语速。

### 设置与兼容性

- 设备 `/setup` 页面和 watcher 本地设置页都提供三段式选择器，并显示当前生效值。
- watcher 接收 `POST /api/setup/style`；设备接收
  `POST /api/match-setup/style`，并复用现有 pending/ack 转发链路。
- 比赛中切换会立即作用于新生成的提醒，不会重播 ESPN 旧事件，也不会重置盘口基线、
  已排队告警或轮询状态。

升级时更新 watcher checkout，并安装 Matchday MOD 1.4.0 即可。本版本不需要修改或
重新烧录官方 Stack-chan host firmware，也不需要更换 TTS 模块。
