# Matchday MOD 1.8.0

[English](#english) | [简体中文](#简体中文)

## English

Device-side visual pack for non-soccer categories. Requires reflashing the
mod; the host firmware stays untouched, and the watcher from 1.7.0 keeps
working (it only sends the new token when configured to).

### Team logos on the probability bar

- Team logos ride the existing flag mechanism as `flag-<code>.png` (22x18
  with alpha), so `pkbar mlb-lad … mlb-phi …` just works. The flag-code
  check widens from country shapes (`fr`, `gb-eng`) to short dashed codes
  (`mlb-lad`).
- New `tools/generate_stackchan_team_logo_pack.py` +
  `config/stackchan_team_logo_pack.json` rasterize logos from the ESPN CDN;
  all 30 MLB clubs ship in this release (~120KB of the 2MB `xs` partition).
- Pairing-registry `display` hints now reference logo codes, so a
  registry-derived MLB watch renders club marks automatically.

### Sport icon on the split

- `pkbar` accepts an optional seventh token `[icon]` choosing the center
  marker: `football` (default) or `baseball`. Unknown names clamp to the
  football, and mods before 1.8.0 silently ignore the extra token.
- The watcher picks the icon from the registry category (`mlb` → baseball)
  or `probability_bar.icon`, and omits the token entirely for football so
  commands stay byte-identical to 1.7.0 on soccer watches.

### Storage note

Assets stay compiled into the mod on purpose: the 2MB `xs` partition is at
~15% after this release, resources render memory-mapped from flash (zero XS
heap cost), and an SD-card path would require a host reflash plus runtime
PNG decoding that does consume RAM. Revisit only if the pack ever approaches
the partition budget.

## 简体中文

面向非足球品类的设备端视觉包。需要重刷 mod；host 固件不动，1.7.0 的
watcher 可以继续用（只有配置了新图标才会发送新参数）。

### 概率条队标

- 队标复用旗帜机制（`flag-<code>.png`，22x18 带透明通道），
  `pkbar mlb-lad … mlb-phi …` 直接可用；旗码校验从国旗形状
  （`fr`、`gb-eng`）放宽为短横线代码（`mlb-lad`）。
- 新增 `tools/generate_stackchan_team_logo_pack.py` 与
  `config/stackchan_team_logo_pack.json`，从 ESPN CDN 栅格化队标；
  本版本内置 MLB 全部 30 队（约 120KB，`xs` 分区共 2MB）。
- 配对注册表的 `display` 提示已改用队标代码，注册表派生的 MLB
  陪看自动显示队徽。

### 分界运动图标

- `pkbar` 接受可选第七个参数 `[icon]` 选择中央图标：`football`
  （默认）或 `baseball`。未知名称回退足球；1.8.0 之前的 mod 会
  静默忽略多余参数。
- watcher 按注册表品类（`mlb` → baseball）或 `probability_bar.icon`
  选择图标；足球时完全省略该参数，足球陪看的指令与 1.7.0 逐字节一致。

### 存储说明

资源刻意保持编译进 mod：本版本后 2MB `xs` 分区仅用约 15%，资源以
flash 内存映射方式渲染（XS 堆零开销）；SD 卡路径反而需要重刷 host
并在运行时解码 PNG（真正消耗 RAM）。仅当资源包逼近分区预算时再议。
